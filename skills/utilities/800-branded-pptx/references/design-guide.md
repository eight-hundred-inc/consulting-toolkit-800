## カラーパレット

### プライマリカラー

| 名称 | HEX | RGB | 用途 |
|------|-----|-----|------|
| ティール | `29BA74` | 41, 186, 116 | メインアクセント、ボタン、強調、アクセントバー |
| ダークグリーン | `1B3928` | 27, 57, 40 | タイトルスライド・セクション開始の背景のみ |

| グレー | `A6A6A6` | 166, 166, 166 | サブテキスト、キャプション |

## タイポグラフィ

### フォントファミリー

| 言語 | フォント | 代替フォント |
|------|----------|--------------|
| 日本語 | Meiryo UI | Meiryo, Yu Gothic UI |
| 英数字 | Century Gothic | Arial, Calibri |

**使い分けルール（同一テキストブロック内でも適用）**:
- 英数字・記号 → `Century Gothic`
- 日本語 → `Meiryo UI`
- 混在テキスト → `splitTextWithFonts()` 関数で自動分割

> **実装**: `splitTextWithFonts()` 関数の詳細は [layout-patterns.md](layout-patterns.md) の「テキスト分割ヘルパー関数」セクションを参照。

### フォントサイズ階層

| 要素 | サイズ | ウェイト | 用途 |
|------|--------|----------|------|
| メインタイトル | 32pt | Regular | タイトルスライドのメインテキスト |
| スライドタイトル | 24pt | Regular | 各スライドのタイトル |
| セクション見出し | 18pt | Bold | セクション内の見出し |
| サブ見出し | 16pt | Bold | カードタイトル等 |
| 本文 | 14pt | Regular | メインコンテンツ |
| 注釈/キャプション | 10-12pt | Regular | 補足テキスト |
| フッター | 8pt | Regular | コピーライト、ページ番号 |

### テキストカラーの使い分け

| 用途 | HEX | 説明 |
|------|-----|------|
| primary | `404040` | メインタイトル、見出し、本文 |
| secondary | `A6A6A6` | サブテキスト、キャプション |
| onDark | `FFFFFF` | ダーク背景上のテキスト |
| accent | `29BA74` | 強調テキスト |

### フォント適用の判断基準

| テキスト内容 | 処理方法 | フォント |
|-------------|----------|----------|
| 英数字のみ | 直接指定 | Century Gothic |
| 日本語のみ | 直接指定 | Meiryo UI |
| 混在 | `splitTextWithFonts()` | 自動分割 |

## レイアウトグリッド

### スライドサイズ

| 項目 | 値 |
|------|-----|
| アスペクト比 | 16:9 |
| 寸法 | 10" × 5.625" |

### マージン

| 位置 | サイズ | 備考 |
|------|--------|------|
| 左右 | 0.5インチ | |
| 上 | 0.25インチ | ヘッダー部分 |
| 下 | 0.4インチ | フッター領域 |

### コンテンツエリア

```
有効幅: 10" - 0.5" × 2 = 9"
有効高さ: 5.625" - 0.25" - 0.4" = 約5"
```

## アイコン使用ガイドライン

アイコンは `react-icons` からPNG形式で生成して使用する。

### 推奨アイコンカラー

| 用途 | HEX | 説明 |
|------|-----|------|
| success | `1B3928` | 成功・完了 |
| error | `E61C56` | エラー・警告 |
| info | `29BA74` | 情報・ヒント |
| neutral | `A6A6A6` | 中立・無効 |

## 避けるべきパターン

| NG | 理由 |
|----|------|
| 過度な装飾 | シンプルでクリーンなデザインを維持 |
| 色の乱用 | プライマリカラー2色 + ニュートラルを基本に |
| テキストの詰め込み | 1スライド1メッセージを心がける |
| センター揃えの本文 | 本文は常に左揃え |
| 影やグラデーション | フラットデザインを基本とする |
| 細すぎる線 | 線は最低0.5pt以上 |
| 小さすぎるフォント | 本文は14pt以上 |
