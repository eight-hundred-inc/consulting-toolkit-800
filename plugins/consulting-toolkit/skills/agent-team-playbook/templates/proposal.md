# 提案フェーズ

前提: Input/ に与件資料が存在

成果物:
- Output/論点・仮説.md — Step 1（論点・仮説の設計）で探索的調査・仮説検証調査を経て作成
- Output/提案書.md — interview-research-proposal スキル準拠（論点・仮説.mdと調査レポートを入力）
- Output/スライド構成_提案.md — slide-structure-designer スキル準拠
- Output/インタビューガイド.md — interview-guide-creator スキル準拠
- Output/報告書骨子.md — report-outline-creator スキル準拠

制約:
- Step 1（論点・仮説の設計）を完了してから提案書を作成する
- 提案書作成後に残りの成果物（スライド構成、インタビューガイド、報告書骨子）を作成
