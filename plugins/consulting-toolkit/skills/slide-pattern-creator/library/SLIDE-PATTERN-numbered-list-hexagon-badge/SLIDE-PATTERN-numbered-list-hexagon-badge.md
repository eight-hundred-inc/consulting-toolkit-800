# SLIDE-PATTERN-numbered-list-hexagon-badge

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

**パターン名：** numbered-list-hexagon-badge
**概要：** 左側に六角形の番号バッジ、右側に見出しと説明文のテキストブロックを横並びにしたリスト行を縦に積み重ねるレイアウト
**適したシーン：** 3〜4項目の分類・対象グループ・特徴を番号付きで整理するスライド。ターゲットセグメントの列挙や複数の要点を同等の重みで並べたい場合に適する

## Structure（構造）

コンテンツエリアは上部のリード文エリアと、下部の番号付きリストエリアで縦に分かれる。リストエリアは各行が「左：六角形番号バッジ ／ 右：テキストブロック（見出し＋説明文）」の2カラム構成で、行間は水平罫線で区切られる。

```yaml
layout: single
zones:
  - role: lead-text
    content: リード文（1〜2行、スライド全体の主旨・前置き）
  - role: list
    content: リスト行 ×3（各行：左に六角形番号バッジ、右にテキストブロック［見出し＋説明文］の2カラム）
reading_order: 上（リード文）→ 下（リスト、上から下）
notes: 各リスト行は水平罫線で区切る
```

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| リード文 | コンテンツエリア上部 | スライド全体の主旨・前置き説明を1〜2行で提示する |
| 六角形番号バッジ | 各行の左端 | 視認性の高い六角形シェイプに番号を入れて項目を識別する |
| 項目見出し | バッジ右側の上段 | 各リスト項目のタイトルをアクセントカラーで強調表示する |
| 項目説明文 | バッジ右側の下段 | 見出しに対する補足・詳細説明を2〜3行で記述する |
| 水平区切り線 | 各行の下端 | リスト行を視覚的に分離して可読性を高める |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-numbered-list-hexagon-badgeのレイアウトで、〔テーマ〕の対象グループを3項目でリストアップしてください。各項目に見出しと2〜3行の説明文を入れてください。デザインはスライドマスターに従ってください。」

**注意点：**
- リスト行は3〜4行が適切。それ以上になると各行の説明文スペースが狭くなる
- 六角形バッジの番号は視認性を保つためにシンプルなアラビア数字を使う
- 項目見出しはアクセントカラーで統一し、全行同じスタイルを維持する
- 各行の説明文量はなるべく揃え、行ごとの高さが均一になるようにする
