# SLIDE-PATTERN-two-lane-pipeline

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスター（表紙・タイトル行・フッター・配色・フォント）はマスター側（branded-pptx / html-artifactテーマ等）で定義されるため、このファイルには含みません。

## Overview
**パターン名：** two-lane-pipeline
**概要：** 上下2本のレーンにそれぞれ「レーンラベル＋4ステップの横フロー＋右端の事例・効果注記」を配置し、下部に変化・帰結を示す3枚のカードを敷く構造。
**適したシーン：** 1つのテーマを2つの段階・領域（例：上流と下流、計画と実行）に分けてそれぞれのプロセスを説明し、最後に読み手にもたらす変化をまとめて提示したいスライド。
**対応する図解指名：** 2レーンパイプライン
**下流実装ヒント：** html-artifact: 概念フロー2段＋注記 ／ pptx: フロー図2段＋事例ボックス

## Structure（構造）
コンテンツエリア最上部に「軸」を説明する導入帯を置き、その下に2本のレーン（上流レーン・下流レーン等）を縦に並べる。各レーンはレーンラベル＋説明見出しのヘッダを持ち、その下に4ステップの横フロー（矢印でつなぐ）と、フロー右端に固定幅の事例・効果注記カードを配置する。最下部には「もたらす変化」の見出しと、3枚横並びのカード（各カード＝見出し＋短文）を置く。

    structure:
      layout: two-lane-stacked
      intro_band:
        label: 軸
        body: 2つのレーンが何を分担するかを説明する導入文
      lanes:
        count: 2
        each:
          lane_label: レーン名（例：上流／下流）
          lane_desc: このレーンで扱う内容の見出し
          flow:
            steps: 4
            each_step: { tag: ステップ名, text: 内容説明 }
            connector: arrow
          case_card:
            width: fixed（レーン右端に固定幅で配置）
            tag: 先行事例
            body: 実績・数値
      bottom:
        label: 御社にもたらす変化（または帰結）
        cards:
          count: 3
          each: { title: 変化の見出し, body: 説明文 }

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 導入帯（軸） | コンテンツエリア最上部 | 2つのレーンがそれぞれ何を担当するのかを一言で定義する |
| レーンラベル | 各レーン左端 | レーンの名称（上流／下流、計画／実行 等）を示すタグ |
| レーン見出し | レーンラベルの隣 | そのレーンで扱うテーマを短く説明 |
| ステップ（4個） | 各レーンのフロー部分 | 処理の流れを「タグ＋説明文」のセットで順に示す |
| 矢印コネクタ | ステップとステップの間 | 処理が左から右へ順番に進むことを視覚的に示す |
| 事例・効果カード | 各レーンの右端固定幅 | そのレーンの実現可能性・効果を示す事例や数値 |
| 変化カード（3枚） | 下部・横並び | 2レーンの取り組みが読み手にもたらす具体的な変化を3つの観点で提示 |

## Usage Guide（AIへの使い方）

> 「SLIDE-PATTERN-two-lane-pipelineのレイアウトで、上流レーンに［4ステップのプロセスと事例］、下流レーンに［4ステップのプロセスと事例］を入れ、下部の変化カード3枚に［この取り組みがもたらす変化3点］を記入してください。デザインはスライドマスターに従ってください。」

**注意点：**
- 情報量が多いパターンのため、各ステップの説明文は1〜2文程度に絞る。詳細を書きすぎるとフローが読みにくくなる。
- 2レーン分＋変化カード3枚を1枚に収めると密度が高くなりやすい。内容が多い場合は「上流レーンのみのスライド」「下流レーンのみのスライド」に分割し、変化カードは別スライドにまとめることを検討する。
- 各レーンのステップ数は4個を標準とする。3個以下で説明が済む場合も4カラムの枠を維持し、空白セルを作らないよう内容を調整する。
