## Overview

**パターン名：** three-col-numbered-card
**概要：** 3列カードの上部にカラーのラベルバンドを持ち、大きな連番数字・下線・見出し・箇条書きで構成されるカードレイアウト。
**適したシーン：** サービスの強み・選ばれる理由・特長を番号付きで3点整理して提示するスライド。順序や優先度を明示したいときに最適。

## Structure（構造）

コンテンツエリアの上部にリード文を置き、その下に3つのカードを横並びで配置する。各カードは上部のラベルバンド（"reason"等の補足ラベル）・大きな番号・下線・見出し・箇条書き本文で縦に積み上げられた構造。カード全体に薄いカラー背景が適用される。

```yaml
layout: three-column-grid
zones:
  - role: lead-text
    content: リード文（カードグループ上部）
  - role: cards
    content: カード ×3（横並び、等幅・等間隔、薄いカラー背景）
    elements:
      - role: label-band
        content: ラベルバンド（カテゴリ・種別を示す短い補足ラベル、最上部）
      - role: number
        content: 連番数字（大きなフォント、中央揃え、視覚的アンカー）
      - role: divider
        content: 仕切り線（番号と見出しの区切り）
      - role: heading
        content: 見出し（2行程度まで、中央揃え）
      - role: body
        content: 箇条書き本文
reading_order: 上（リード文）→ 左から右（カード）、各カードは上から下（ラベルバンド→番号→仕切り線→見出し→本文）
notes: 3カードは等幅・等間隔で横並びにする。番号下の仕切り線はブランドカラーで引く。箇条書き本文エリアはカードとは異なる薄いカラー背景を適用する。
```

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| リード文 | カードグループ上部 | スライド全体の補足説明・文脈を示すテキスト |
| ラベルバンド | カード最上部 | カードのカテゴリ・種別を示す小さな補足ラベル帯 |
| 連番数字 | バンド直下、中央 | カードの順序を示す大きな数字（視覚的アンカー） |
| 仕切り線 | 番号下 | 番号と見出しを区切る水平線 |
| 見出し | 仕切り線直下、中央揃え | カードの主題を示すテキスト（2行程度まで） |
| 箇条書き本文 | 見出し下、薄いカラー背景エリア | 各項目の具体的な説明を箇条書きで列挙 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-three-col-numbered-cardのレイアウトで、サービスが選ばれる3つの理由（各理由の詳細説明付き）を提示するスライドを作成してください。デザインはスライドマスターに従ってください。」

**注意点：**
- ラベルバンドには "reason" "point" "feature" などの短い補足語を入れる
- 番号は大きなフォントサイズで中央配置し、視覚的なアンカーとして機能させる
- 番号下の仕切り線はブランドカラーで引く
- 見出しは2行以内に収め、中央揃えにする
- 箇条書き本文エリアにはカードとは異なる薄いカラー背景を適用する
- 3カードは等幅・等間隔で横並びにする
