# SLIDE-PATTERN-stage-table-with-points

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。

## Overview

**パターン名：** stage-table-with-points
**概要：** コンテンツエリアを左3/4と右1/4に分け、左に4段階の成熟度・進化ステージ横並び矢印マトリクス（現在地・到達点のマーカー付き）、右にキーポイント箇条書きを配置する成熟度モデル型レイアウト。
**適したシーン：** BCG成熟度モデルや進化ステージ論に自社の現在地・到達点を重ねて位置づけを説明するスライド。「現在地はStage 1、目指すはStage 3」のような段階的な進化ロードマップを示す場面に最適。

## Structure（構造）

コンテンツエリアを左75%（マトリクス）と右25%（ポイント）に分割。左は上部にマーカー行（「現在地」「到達点」等）、中段に4つの矢印形状のステージ、下段に「特徴」「意味合い」の2行の説明テキスト。右は箇条書き3〜4項目。

    structure:
      layout: left-4stage-matrix-right-points
      left:
        width: 75%
        rows:
          - name: "マーカー行（現在地▼、到達点▼ 等）"
          - name: "ステージ矢印（4個・横並び）"
          - name: "特徴の説明（各ステージ下）"
          - name: "意味合いの説明（各ステージ下）"
      right:
        width: 25%
        content: "キーポイント箇条書き 3〜4項目"

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| マーカー行 | 左エリア最上部 | 「現在地」「Layer A到達点」「Layer B到達点」等をステージ位置に紐づけ |
| ステージ矢印 | 中段（横並び4個） | Stage 1〜4を矢印形状で表現。現在地・到達点は塗りや網掛けで強調 |
| 特徴行 | 各ステージ矢印の下 | ステージの一般的特徴を1〜2行で記述 |
| 意味合い行 | 特徴行の下 | 自社（発表主体）にとっての意味を1〜2行で記述 |
| キーポイント | 右カラム | 図から読み取れる示唆・ポイントを箇条書きで補強 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-stage-table-with-pointsのレイアウトで、左に4段階の成熟度モデル（矢印形状で横並び）を配置し、現在地と到達点をマーカーで示し、右にキーポイント3項目を配置してください。デザインはスライドマスターに従ってください。」

**注意点：**
- ステージは4段階が標準（BCG成熟度モデル等の外部フレームワーク準拠）
- 各ステージの矢印形状は同じ幅にし、現在地・到達点のみ塗りや網掛けで差別化する
- 右のキーポイントは「読み取り所」を絞り込む — 図から自明なことを繰り返さない
