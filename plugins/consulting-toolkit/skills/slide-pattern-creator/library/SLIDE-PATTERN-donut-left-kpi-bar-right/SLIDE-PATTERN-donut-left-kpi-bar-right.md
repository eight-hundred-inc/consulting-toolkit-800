# SLIDE-PATTERN-donut-left-kpi-bar-right
このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。タイトルエリアを除いたコンテンツ領域のみを対象とし、スライド全体のテンプレートではありません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

左側（40%幅）に大きなドーナツグラフと中央数値を配置し、右側（60%幅）を上下に分割して上段に3つのアイコンKPI、下段に横棒グラフを並べるダッシュボード型レイアウト。ひとつのスライドで複数種類の数値情報を同時に表示したい場合に適している。

- **適用場面:** KPIダッシュボード、業績サマリー、定量レポート
- **情報密度:** 高（グラフ×2種＋KPI×3のコンパクト集約）
- **方向性:** 視覚的な達成率と複数指標の同時提示

## Structure

```yaml
layout: donut-left-kpi-bar-right
zones:
  - role: donut-chart
    width: 40%
    content: エリアラベル＋ドーナツグラフ（2分割）＋中央数値＋凡例2項目
  - role: right-column
    width: 60%
    elements:
      - role: kpi-band
        content: アイコン付きKPI × 3（横並び）
      - role: bar-chart
        content: 横棒グラフ（ラベル＋バー＋数値） × 4
reading_order: 左（ドーナツグラフ）→ 右上（KPI帯）→ 右下（横棒グラフ）
notes: 右カラムは上下二段に分割し、上段にKPI帯、下段に横棒グラフを配置する
```

## Elements

### 左エリア（ドーナツグラフゾーン）
| 要素 | 説明 |
|------|------|
| エリアラベル | 「ドーナツグラフ」等の小文字ラベル |
| ドーナツグラフ | 2分割構成の円グラフ |
| 中央数値 | 達成率や主要値を太字で表示 |
| 凡例行 | 2項目を横並びで表示 |

### 右上エリア（KPIバンド）
| 要素 | 説明 |
|------|------|
| アイコン円 | 各KPIを表すアイコンプレースホルダー |
| 数値 | 太字で表示 |
| 単位 | 数値に添える単位表記 |
| ラベル | KPI項目名 |

### 右下エリア（横棒グラフ）
| 要素 | 説明 |
|------|------|
| エリアラベル | 棒グラフのカテゴリ名 |
| ラベル列 | 各バーの項目名 |
| バー背景 | バーの下地（トラック） |
| バー本体 | 値に応じた可変幅のバー |
| 数値テキスト | バー末尾に数値を表示 |

## Usage Guide

- 左のドーナツは「全体達成率」など単一の主要KPIを表示するのに最適
- 右上KPIは同じ軸（件数、金額、人数など）の異なる切り口を並べる
- 右下横棒グラフはカテゴリ別内訳や部門比較に使用する
- 凡例は2項目以内に絞り、簡潔に保つ
- 配色はスライドマスターのブランドカラーに従う
