# SLIDE-PATTERN-numbered-question-list

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスター（表紙・タイトル行・フッター・配色・フォント）はマスター側（branded-pptx / html-artifactテーマ等）で定義されるため、このファイルには含みません。

## Overview
**パターン名：** numbered-question-list
**概要：** 論点・問いを番号チップ付きで列挙するリスト。左に大論点（カテゴリチップ）、右に番号チップ＋問い文を置く2階層構成にも対応する。
**適したシーン：** 検討すべき論点・調査で答えるべき問い・アジェンダなどを構造的に列挙したいスライド。論点をカテゴリ（大論点）ごとにグルーピングし、その下に具体的な問い（小論点）を並べたい場面に向く。
**対応する図解指名：** 番号付き論点リスト
**下流実装ヒント：** html-artifact: Priority List / Q&A Card ／ pptx: 番号チップ＋論点行

## Structure（構造）
コンテンツエリアを大論点ごとの行グループに分割する。各グループは、左に大論点のカテゴリチップ、右にそのカテゴリに属する小論点（番号チップ＋問い文）を縦に並べる2階層構成とする。番号は全体を通した通し番号（①②③…）とし、左のカテゴリチップは対応する小論点群と縦位置を揃える。大論点が不要な場合は左カテゴリを省き、番号チップ＋問い文の1階層フラットリストとしても使える。

    structure:
      layout: numbered-question-list
      hierarchy: 2-level (optional 1-level)
      groups:
        each:
          major_topic: 大論点（左のカテゴリチップ）
          questions:
            - chip: 通し番号（①②…）
              text: 小論点（問い文）
      alignment: 左カテゴリチップと右の小論点群を縦位置で揃える
      fallback: 大論点が不要なら左を省き番号チップ＋問い文の1階層リストにする

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| カテゴリチップ（大論点） | 各グループ左 | 小論点をまとめる上位の論点カテゴリを示す。2階層構成の上位 |
| 番号チップ | 各小論点の左 | 論点・問いの通し番号。参照や議論の順序づけに使う |
| 問い文（小論点） | 番号チップの右 | 検討すべき具体的な論点・問いを1文で示す |
| グループの縦揃え | 左カテゴリ⇔右小論点群 | どの小論点がどの大論点に属するかを縦位置の対応で示す |

## Usage Guide（AIへの使い方）

> 「SLIDE-PATTERN-numbered-question-listのレイアウトで、大論点を［カテゴリA／カテゴリB／カテゴリC］に分け、各カテゴリの下に番号付きの小論点（問い文）を並べてください。番号は全体通し番号にしてください。デザインはスライドマスターに従ってください。」

**注意点：**
- 番号は全体を通した通し番号にする。カテゴリごとに①から振り直すと論点間の参照がぶれる。
- 問い文は「〜か？」の疑問形または論点として明確な形にする。単なる名詞（例：「市場規模」）だけにすると論点が曖昧になる。
- 大論点（カテゴリ）が2つ以下、または全体で問いが4件以下なら、無理にグルーピングせず1階層フラットリストにする。
