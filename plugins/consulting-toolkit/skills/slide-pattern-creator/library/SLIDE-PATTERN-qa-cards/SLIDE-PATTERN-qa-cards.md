# SLIDE-PATTERN-qa-cards

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスター（表紙・タイトル行・フッター・配色・フォント）はマスター側（branded-pptx / html-artifactテーマ等）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview
**パターン名：** qa-cards
**概要：** 問いを4枚のカードに分けて2×2グリッドで並べ、うち1枚のみを最優先の問いとして枠・塗りで強調するレイアウト。
**適したシーン：** インタビューやディスカッションの冒頭で「今日確認したい問い」を複数提示し、優先順位を明確にしたいスライド。
**対応する図解指名：** Q&Aカード
**下流実装ヒント：** html-artifact: Q&A Card ／ pptx: 2×2グリッドの角丸矩形、最優先カードのみ塗りつぶし（ダーク背景）＋バッジで区別

## Structure（構造）
コンテンツエリアは2列×2行のグリッドで、4枚の問いカードを均等に配置する。各カードはQ番号バッジ（例：問い1〜4）と問い本文で構成され、任意でカードに優先度フラグ（例：「最優先」）を付与できる。4枚のうち1枚（通常は問い1）だけを視覚的に強調して最優先であることを示し、残り3枚は同一トーンの通常カードとして扱う。

```yaml
layout: grid-2x2
zones:
  - role: card
    content: 問いカード ×4（2×2 グリッド、均等配置）。各カードは Q番号バッジ（例：問い1〜4）＋問い本文（1〜3文程度）／任意で最優先カードのみ優先度フラグ（例：最優先）を付与
reading_order: 左上 → 右上 → 左下 → 右下
notes: 4枚中1枚のみ強調表示（highlighted）とし、他3枚は通常表示（normal）とする
```

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| Q番号バッジ | 各カード左上 | 問いの通し番号を示すラベル（例：問い1〜問い4） |
| 優先度フラグ | 最優先カードのバッジ横のみ | その問いが今回の議論で最も重要であることを明示するタグ |
| 問い本文 | 各カード中央 | 相手に確認したい具体的な問いの内容 |
| 強調カード | グリッド中の1枚のみ | 濃色背景または太枠で他の3枚と差別化し、最優先の問いに視線を集める |

## Usage Guide（AIへの使い方）

> 「SLIDE-PATTERN-qa-cardsのレイアウトで、問い1〜4の内容［各問いの本文］を入れ、問い［番号］を最優先として強調してください。デザインはスライドマスターに従ってください。」

**注意点：**
- 強調するカードは必ず1枚に限定する。2枚以上を強調すると優先順位が伝わらなくなる。
- 問いの本文は1〜3文程度に収める。長文になるとカードからはみ出すため、複数の問いに分割することを検討する。
- 問いが3枚以下の場合は2×2グリッドの1枠を空けず、3枚グリッドやカード幅を広げるなど別パターンでの表現も検討する。
