# SLIDE-PATTERN-icon-left-text-list

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

**パターン名：** icon-left-text-list
**概要：** 3〜4つの項目を縦に並べ、各行の左にアイコン（画像・シンボル）、右に見出しと本文を配置する横展開リストレイアウト
**適したシーン：** サービスの機能説明、手順や前提条件の説明、特徴を視覚的に整理したい場面

## Structure（構造）

コンテンツエリアを3〜4行に分割し、各行は左側のアイコンエリアと右側のテキストエリアで構成される。行間は薄い区切り線で分ける。各行の高さは均等。

```yaml
layout: icon-left-text-rows
zones:
  - role: row
    count: 3
    elements:
      - role: icon
        width: 約10%
        position: left
        content: アイコン（画像またはシンボル）
      - role: text
        position: right
        content: 見出し（H2・カラー）＋本文
    divider: 行区切り線（下端）
reading_order: 上から下（行1→行2→行3）
notes: 各行の高さは均等にする
```

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| アイコン | 各行左側 | 項目をシンボルで視覚化 |
| 見出し（H2） | 右側上部（アクセントカラー） | 項目のキーワード |
| 本文 | 見出し下 | 詳細説明（2〜4行） |
| 行区切り線 | 行間 | 項目の境界を明確にする |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-icon-left-text-listのレイアウトで、3つの[テーマ]をアイコン付きの縦リストで説明してください。デザインはスライドマスターに従ってください。」

**注意点：**
- アイコンは実際の画像・SVG・シンボル文字で表現する
- 見出しはアクセントカラー（ブランドカラー）にすると視認性が上がる
- 各行の本文テキスト量を均等に揃えるとバランスが良い
- このパターンはnumbered-list-with-bodyと似ているが、数字バッジの代わりにアイコン画像を使う点が特徴
