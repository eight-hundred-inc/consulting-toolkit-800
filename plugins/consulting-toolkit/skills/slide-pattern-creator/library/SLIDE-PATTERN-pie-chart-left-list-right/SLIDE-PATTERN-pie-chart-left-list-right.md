# SLIDE-PATTERN-pie-chart-left-list-right

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

**パターン名：** pie-chart-left-list-right
**概要：** コンテンツエリアを左右2カラムに分割し、左側に円グラフ（中央に数値やラベルを表示するドーナツ型も可）、右側に凡例または詳細リストを縦に並べるレイアウト。
**適したシーン：** 構成比・割合の説明、アンケート結果、市場シェアの内訳説明、予算配分の内訳など数値の比率を見せながら詳細も伝えたいスライド。

## Structure（構造）

コンテンツエリアを左右2カラムに分割する。左カラム（幅約45%）には円グラフまたはドーナツチャートを配置し、中央にキャッチアップ数値やラベルを表示する。右カラム（幅約55%）には各区分の色付きラベル・名称・数値・説明を縦に並べたリストを配置する。

    layout: two-column
    zones:
      - role: chart
        width: 45%
        content: 円グラフ／ドーナツチャート（中央ラベルは任意）
      - role: legend
        width: 55%
        content: 凡例リスト（カラードット＋ラベル＋数値＋説明）を縦に並べる
    reading_order: 左 → 右
    notes: 区分数は2〜6個が見やすい（7つ以上は「その他」にまとめる）。右カラムの凡例はグラフの色と対応させる

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 円グラフ / ドーナツグラフ | 左カラム中央 | 各区分の比率を視覚的に示す主要チャート |
| 中央ラベル | ドーナツグラフの中央 | 合計値・キーワードなど最も伝えたい数値やテキストを大きく表示（省略可） |
| 凡例カラードット | 右カラム各アイテム左端 | グラフの各区分に対応する色付き丸で凡例を示す |
| 凡例ラベル | ドットの右横 | 各区分の名称を表示 |
| 数値バッジ | 凡例ラベルの右または下 | 各区分のパーセントや実数値を強調表示 |
| 説明テキスト | 数値の下 | 各区分の補足説明（省略可） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-pie-chart-left-list-rightのレイアウトで、顧客満足度アンケートの回答結果を円グラフで示しながら、右側に各選択肢の割合と補足コメントを列挙するスライドを作成してください。デザインはスライドマスターに従ってください。」

**注意点：**
- 区分数は2〜6個が見やすい。7つ以上はグラフが小さくなるため「その他」にまとめる
- ドーナツ型を使う場合は中央ラベルに合計値やキーメッセージを入れると情報密度が上がる
- 右カラムの凡例リストはグラフの色と必ず対応させる
- 右カラムに説明テキストを入れると縦スペースが不足しやすいため、説明は1行以内に収める
