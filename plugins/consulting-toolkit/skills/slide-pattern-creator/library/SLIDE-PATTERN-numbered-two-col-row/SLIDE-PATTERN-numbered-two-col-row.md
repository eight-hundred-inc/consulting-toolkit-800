# SLIDE-PATTERN-numbered-two-col-row

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

**パターン名：** numbered-two-col-row
**概要：** 3〜4項目を横幅全体のロー（行）として縦に積み、各ローを左カラム（番号バッジ＋見出し）と右カラム（箇条書き詳細）の2列に分割したレイアウト
**適したシーン：** 施策・特徴の詳細説明、左に概念・右に詳細の見せ方、提案内容の構造化

## Structure（構造）

コンテンツエリアを3〜4行に分割し、各行は左右2カラムに区切られる。左カラムに番号バッジと見出し、右カラムに箇条書き（2〜3点）を配置する。行間は水平線で区切る。

```yaml
layout: row-based-two-column
zones:
  - role: row
    content: リスト行 ×3〜4（各行は左右2カラム、行間は水平線で区切る）
    elements:
      - role: left
        width: 40%
        content: 番号バッジ＋見出し（H2）
      - role: right
        width: 60%
        content: 箇条書き（2〜3点）
reading_order: 上から下（行順）。各行内は左→右
notes: 右カラムの箇条書きは1項目あたり1〜2行以内に収める
```

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 番号バッジ | 各行左端 | 項目番号を視覚的に示す |
| 見出し（H2） | バッジ右 | 各行の主テーマ・キーワード |
| 箇条書き | 各行右カラム | 見出しを具体化する詳細ポイント（2〜3点） |
| 水平区切り線 | 行間 | 行の境界を明確に示す |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-numbered-two-col-rowのレイアウトで、3つの施策について左に見出し・右に詳細ポイントを2〜3個ずつ入れてください。デザインはスライドマスターに従ってください。」

**注意点：**
- 右カラムの箇条書きは1項目あたり1〜2行以内に収める
- 4行以上になる場合は行が窮屈になるため、行数を減らすかレイアウト調整を検討する
- 左カラムの見出しは太字・アクセントカラーで強調するとメリハリが出る
