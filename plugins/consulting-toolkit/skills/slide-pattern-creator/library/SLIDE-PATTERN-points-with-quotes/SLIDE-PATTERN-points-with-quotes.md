# SLIDE-PATTERN-points-with-quotes

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスター（表紙・タイトル行・フッター・配色・フォント）はマスター側（branded-pptx / html-artifactテーマ等）で定義されるため、このファイルには含みません。

## Overview
**パターン名：** points-with-quotes
**概要：** 左に番号付きの特筆ポイントを列挙し、右にそれを裏付ける発言者付きの引用コメントを並べる2カラム構成。
**適したシーン：** インタビュー・ヒアリング・アンケートの示唆整理。定性的な発見（左）とその一次情報（右の生声）を同一スライド上で対応させて見せたいとき。
**対応する図解指名：** 特筆ポイント＋引用2カラム
**下流実装ヒント：** html-artifact: fact-list（番号付き）＋引用カード ／ pptx: 2カラム＋引用ボックス（左に番号チップ＋太字見出し、右にコメント枠）

## Structure（構造）
上部に全幅のリード文（このスライドで言いたいことの要約1文）を置き、その下を左右2カラムに分割する。左カラムは番号チップ＋太字見出し＋短文の「特筆ポイント」を縦に3〜4件、右カラムは発言者ラベル／アイコン付きの「引用コメント」を2〜3件並べる。左右は概ね等幅（1:1）。

```yaml
layout: two-column-with-lead
areas:
  - name: lead
    role: 全幅の要約文（1文）
    position: top-full-width
  - name: points
    role: 番号付き特筆ポイント列
    position: left
    width: 50%
    items:
      - number-chip   # 丸番号
      - bold-heading  # 太字見出し
      - short-text    # 補足の短文
    count: 3-4
  - name: quotes
    role: 引用コメント列（発言者付き）
    position: right
    width: 50%
    items:
      - speaker-avatar # 発言者アイコン
      - speaker-label  # 発言者ラベル
      - quote-text     # コメント本文
    count: 2-3
```

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|------|------|------|
| リード文 | 上部・全幅 | このスライドの結論を1文で示す |
| 番号チップ | 左カラム各行の先頭 | ポイントの順序・件数を明示する |
| 太字見出し | 左カラム・番号チップの右 | 各ポイントの要点を短く言い切る |
| 補足短文 | 左カラム・見出しの下 | 見出しを1〜2行で補足する |
| 発言者アイコン | 右カラム各コメントの先頭 | 発言者を視覚的に区別する（顔写真／イニシャル） |
| 発言者ラベル | 右カラム・アイコン脇 | 属性や役職など発言者の立場を示す |
| コメント本文 | 右カラム | 左のポイントを裏付ける一次情報（生声） |

## Usage Guide（AIへの使い方）
プロンプト例：「インタビュー結果から特筆すべき点を3つ抽出して左に番号付きで整理し、右にそれぞれを裏付ける対象者の発言を引用コメントとして配置して。上部に全体の結論を1文で。」

注意点：
- 左のポイントと右の引用は件数を無理に一致させない。ポイントは3〜4件、引用は代表的なものを2〜3件に絞る（右を詰め込むと読めなくなる）。
- 引用は要約せず、話し言葉のニュアンスを残した短い抜粋にする。長い場合は末尾を「…」で省略する。
- 一次情報（引用）が無い場合は右カラムを外し、左の特筆ポイントを全幅の fact-list として使ってよい。
