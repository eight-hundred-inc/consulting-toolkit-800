# SLIDE-PATTERN: vertical-step-three-col

## 概要

縦に並ぶステップを「バッジ列（左）・ステップ名列（中）・詳細説明列（右）」の3列で表現するフローレイアウト。各行が点線で縦に連結される。

## 適したシーン

採用フロー・審査プロセス・実施ステップなど、5〜6段階の工程を「バッジ＋見出し＋詳細箇条書き」で詳しく説明したいとき。

## Structure

```yaml
layout: three-column-vertical
rows: 5
zones:
  - role: step-badge
    width: 15%
    content: 番号バッジ（STEP1, STEP2等）
  - role: step-name
    width: 25%
    content: ステップ名（見出し）
  - role: detail
    width: 60%
    content: 詳細箇条書き（1〜2項目）
row_connector: 各行を結ぶ縦の連結線
reading_order: 左 → 中央 → 右（各行は上から下）
notes: 5〜6行が最適（3〜4行の場合は行の高さを増やし余白を設ける）。右列の詳細は1〜2項目にまとめる。
```

## Elements

| 要素 | 配置 | 役割 |
|---|---|---|
| ステップバッジ | 左列・各行 | 「STEP1」「STEP2」等の番号バッジ |
| 縦連結線 | バッジ間 | ステップの流れを示す縦の連結線 |
| ステップ名 | 中列・各行 | 各ステップの名称（見出し） |
| 詳細箇条書き | 右列・各行 | ステップの詳細説明（1〜2項目・箇条書き） |
| リード文 | 全体上部 | フロー全体の説明（1行） |

## Usage Guide

- 左バッジはグラデーション（上→下で色変化）にするとフローの流れが視覚化される
- 中列の見出しと右列の詳細の間に薄い縦線を入れると整理感が増す
- 5〜6ステップが最適。3〜4ステップの場合は行の高さを増やして余白を設ける
- バッジの背景色を段階的に濃くすることで進捗感が表現できる
- 右列の詳細は1〜2項目にまとめ、詳細を詰め込みすぎない
