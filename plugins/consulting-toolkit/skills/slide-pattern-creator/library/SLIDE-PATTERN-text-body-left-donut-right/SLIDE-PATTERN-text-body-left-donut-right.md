# SLIDE-PATTERN-text-body-left-donut-right

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライド生成AIや人間のデザイナーが一貫したレイアウトを再現できるよう、構造・要素・使用方法を定義します。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

| 項目 | 内容 |
|------|------|
| パターン名 | text-body-left-donut-right |
| 別名・通称 | 左テキスト＋右ドーナツチャート |
| 用途 | 左側（55%）に説明テキスト＋2カラムのポイントリスト、右側（45%）に大きなドーナツグラフと中央数値を配置するチャート説明レイアウト |
| 適したコンテンツ | 利用率・割合・構成比の説明、アンケート結果、市場シェア分析、達成率の報告 |
| レイアウト種別 | 左右分割（テキスト55%＋グラフ45%） |

## Structure

コンテンツエリアを左のテキストゾーンと右のドーナツグラフゾーンに2分割する構造です。

```yaml
layout: text-body-left-donut-right
zones:
  - role: text
    width: 55%
    content: 見出し＋区切り線＋説明テキスト（2行）＋2カラムリスト（各カラム3〜5項目、列ラベル付き）
  - role: donut_chart
    width: 45%
    content: エリアラベル＋ドーナツグラフ（1〜2分割、中央に数値・説明テキスト）＋凡例
reading_order: 左（テキスト）→ 右（ドーナツグラフ）
notes: ドーナツグラフは2〜3分割が視認性の上限（それ以上は円グラフや横棒グラフを検討）。中央数値は大きく強調し一目で認識できるようにする。
```

## Elements

### コンテンツエリア全体
- 左右2ゾーンで構成される横並びレイアウト

### 左エリア（55%）
- 見出し（グラフの結論を一言で表すキーメッセージ）
- 区切り線
- 説明テキスト（2行）
- 2カラムリスト：列ラベル＋箇条書き項目（各カラム3〜5項目、「・テキスト」形式）

### 右エリア（45%）
- エリアラベル
- ドーナツグラフ（1〜2分割の割合表示）
- 中央テキスト（主要な数値・説明を2〜3行）
- 凡例（各色が示す内容の説明）

---

## Usage Guide

### 適した場面
- 単一の割合・比率データをグラフで強調し、その背景を左側で補足説明したい場合
- 「60%の顧客が〇〇」などのインパクトある数値を大きく見せたい場合
- 定量データ（グラフ）と定性データ（リスト）を同一スライドで組み合わせたい場合

### 入れるべき情報
| エリア | 記載内容 |
|--------|--------|
| 左：見出し | グラフの結論を一言で表すキーメッセージ |
| 左：説明テキスト | グラフデータの背景・文脈・補足（2行以内） |
| 左：2カラムリスト | 補足的な詳細ポイント（各カラム3〜5項目） |
| 右：ドーナツグラフ | メインの割合データ（1〜2分割） |
| 右：中央テキスト | 主要な数値と単位（大きく表示） |
| 右：凡例 | グラフの各色が何を示すか |

### 注意点
- ドーナツグラフは2〜3分割が視認性の上限（それ以上は円グラフや横棒グラフを検討）
- 中央数値は大きく強調して一目で認識できるようにする
- 左の2カラムリストの項目数は左右で揃えると整然として見える
- 説明テキストはグラフの解釈をサポートする内容にする
