## Overview

**パターン名：** numbered-two-col-row-with-accent
**概要：** 左側に番号ブロック（斜めカット形状）、右側に見出しと箇条書きを持つテキストエリア、行右端に縦アクセントバーを配置したリスト行レイアウト
**適したシーン：** 3〜4つのプロセス・手順・フェーズを番号付きで説明するスライド。各項目に複数の詳細箇条書きがあり、視覚的なメリハリを出したい場合に最適

## Structure（構造）

コンテンツエリアは番号付きリスト行を縦に積み重ねた構造。各行は「左：番号ブロック（斜めカット形状） ／ 中央：テキストエリア（見出し＋箇条書き）／ 右：縦アクセントバー」の3カラムで構成される。行間は十分な余白で区切られる。

```yaml
zones:
  - role: list
    content: リスト行 ×3（各行：左に番号ブロック、中央にテキストエリア、右に縦アクセントバーの3カラム構成。行間は十分な余白）
    elements:
      - role: number-block
        content: 番号ブロック（正方形、右端に斜めカット形状、大きな番号01/02/03）
      - role: text-area
        content: 見出し（アクセントカラーのアンダーライン付き）＋箇条書き2〜3項目
      - role: accent-bar
        content: 縦アクセントバー（行右端の区切り線）
reading_order: 上から下（行順）。各行内は左→中央→右

```

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 番号ブロック | 各行の左端 | 正方形ブロックに大きな番号（01/02/03）を表示。右端に斜めカット形状を持つ |
| 斜めカット形状 | 番号ブロックの右端 | 番号ブロックとテキストエリアの境界を示す |
| 項目見出し | テキストエリアの上段 | 各行のタイトルをアクセントカラーのアンダーラインで強調表示する |
| 箇条書きリスト | テキストエリアの中下段 | 見出しの詳細を2〜3項目の箇条書きで列挙する |
| 縦アクセントバー | 各行の右端 | アクセントカラーの縦線で行の右端を区切る |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-numbered-two-col-row-with-accentのレイアウトで、〔プロセス・手順名〕を3ステップで説明してください。各ステップに見出しと2〜3個の箇条書き詳細を入れてください。デザインはスライドマスターに従ってください。」

**注意点：**
- 番号は「01」「02」「03」のようにゼロパディングした2桁表記が視覚的にきれい
- 斜めカット形状は番号ブロックとテキストエリアの境界を示す要素として配置する
- 各行の箇条書きは2〜3項目に収める。それ以上になると行が縦に伸びすぎて全体バランスが崩れる
- 縦アクセントバーはデザインテーマのアクセントカラーに合わせる
