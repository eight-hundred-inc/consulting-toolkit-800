# SLIDE-PATTERN-logo-grid

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。

## Overview

**パターン名：** logo-grid
**概要：** クライアント企業のロゴを均等サイズで6〜8列×4〜6行のグリッドに配置する取引実績・パートナー紹介レイアウト。ロゴのみを大量に並べることで「これだけの企業と取引実績がある」ことを視覚的に訴求する。
**適したシーン：** 会社紹介の取引実績ページ、パートナー企業一覧、導入事例のロゴ一覧、認定資格や受賞歴のバッジグリッド。

## Structure（構造）

コンテンツエリアを均等な格子（6〜8列×4〜6行）に分割し、各セルに1つのロゴを配置。セル間には十分な余白（gap: 30〜40px）を取り、ロゴが窮屈にならないようにする。ロゴのサイズは各セルで統一（幅120〜150px程度）。背景は白。ロゴの縦横比が異なる場合は、セル内で中央揃えで配置する。

    structure:
      layout: logo-grid
      rows: 4-6
      columns: 6-8
      cell:
        width: "均等"
        content: "1ロゴ（幅120〜150px程度）"
        alignment: "中央揃え"
        background: "透明または白"
      gap: "30〜40px"

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| ロゴセル | グリッドの各セル | 1社のロゴを表示。全セルで大きさ・位置を統一 |
| ロゴ画像 | 各セルの中央 | 企業ロゴ（PNGまたはSVG） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-logo-gridのレイアウトで、24〜30社のクライアントロゴを6列×5行のグリッドに配置してください。ロゴサイズは全て統一し、セル間に十分な余白を取ってください。デザインはスライドマスターに従ってください。」

**注意点：**
- ロゴの並び順は「業界別」「取引時期別」「五十音順」等の一貫したルールを選ぶ
- ロゴが横長・縦長で大きさが違う場合、視覚的な大きさ（面積）を揃える調整を各ロゴに施す
- あまりに多くのロゴを詰め込むと1つ1つが判読できなくなる。最大でも30社程度が目安
