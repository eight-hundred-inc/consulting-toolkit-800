# SLIDE-PATTERN-profile-cards

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。

## Overview

**パターン名：** profile-cards
**概要：** コンテンツエリアに3人のプロフィール（顔写真＋名前バッジ＋経歴テキスト）を横並びで配置する経営陣・キーメンバー紹介レイアウト。各カードは円形写真、緑バッジで役職＋氏名、その下に経歴を6〜10行の縦長テキストで記載する。
**適したシーン：** 会社紹介の経営メンバー紹介、プロジェクトのキーメンバー紹介、登壇者紹介スライド。3人が横並びでほぼ等価に紹介される場面に最適。

## Structure（構造）

コンテンツエリアを3等分し、各カラムに1人のプロフィールを縦配置。上から「円形写真」「役職＋氏名バッジ（アクセントカラー塗り）」「経歴テキスト（Text Subカラー、10〜13行）」の順で並べる。カード間は罫線なしで余白のみで分離する。

    structure:
      layout: three-column-profile
      columns: 3
      profile:
        - "円形写真（180〜200px直径）"
        - "アクセントカラーバッジ（役職・氏名）"
        - "経歴テキスト（Text Subカラー・10〜13行）"

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 顔写真 | カード最上部 | 円形にトリミングした顔写真（明るめの背景、ビジネスカジュアル） |
| 役職＋氏名バッジ | 写真の下 | アクセントカラー（グリーン）の塗りに白文字で「役職 / 氏名」を1行に表示 |
| 経歴テキスト | バッジの下 | 略歴・経験・現職を10〜13行の長文で記述 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-profile-cardsのレイアウトで、3人の経営メンバーのプロフィールを横並びで配置してください。各人には円形写真、緑バッジの役職＋氏名、経歴テキスト10行を含めてください。デザインはスライドマスターに従ってください。」

**注意点：**
- 3人が視覚バランス的に最適。2人だと余白が空きすぎ、4人以上は写真が小さくなる
- 経歴テキストの長さは3人でだいたい揃える（極端に長短のばらつきを避ける）
- 写真がない場合はイニシャル入りの円形プレースホルダーで代替可
