# SLIDE-PATTERN-eight-card-2x4-grid
このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。タイトルエリアを除いたコンテンツ領域のみを対象とし、スライド全体のテンプレートではありません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

2行×4列の計8カードをグリッド配置し、各カードに番号バッジ・見出し・説明テキストを収める大容量情報カードレイアウト。多数の施策・機能・特長・手順を1スライドでコンパクトに一覧表示したい場合に適している。

- **適用場面:** 施策一覧、機能紹介、取り組み概要、チェックリスト
- **情報密度:** 高（8項目×各3要素の同時表示）
- **方向性:** 網羅感・一覧性の提示

## Structure

```yaml
layout: grid-2x4
zones:
  - role: card
    content: 番号バッジ＋見出し＋説明テキスト（2行）
    count: 8
    arrangement: 2行×4列グリッド
reading_order: 上段左→右（①→④）→ 下段左→右（⑤→⑧）
notes: 全カードでテキスト量・行高さを揃える。8カードすべてに内容が入ることが前提（空カードが生じる場合は別パターンを検討）
```

## Elements

### グリッドコンテナ
| 要素 | 説明 |
|------|------|
| 列数 | 4列 |
| 行数 | 2行 |
| 全体配置 | コンテンツエリア全体を縦方向に埋める |

### 各カード
| 要素 | 説明 |
|------|------|
| カード（枠面） | 番号バッジ・見出し・説明テキストを縦に並べる |

### カード内要素
| 要素 | 説明 |
|------|------|
| 番号バッジ | 円形のバッジ |
| バッジ数字 | 太字の番号 |
| 見出し | 太字の見出し |
| 説明テキスト | 2行の説明文 |

## Usage Guide

- 8カードは「施策1〜8」「機能A〜H」「ステップ01〜08」のように統一した命名で揃える
- 見出しは10文字以内、説明テキストは30〜40文字（2行）を目安にすると収まりが良い
- 番号バッジの背景色をカテゴリごとに変えることで、グループ分けを視覚化できる
- カードの行高さが揃うよう、説明テキストの文字数を各カードで均等に調整する
- 8カードすべてに内容を記述することが前提。空カードが生じる場合は別パターンを選ぶ
