# SLIDE-PATTERN-phase-roadmap

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。

## Overview

**パターン名：** phase-roadmap
**概要：** コンテンツエリア上部に4つのフェーズを右向き矢印で横並びに配置し、フェーズごとの「ゴール」を書き、その下に「アクション（レイヤー別）」「実現する価値」の複数行を横串で並べる、複年度・複層のロードマップレイアウト。
**適したシーン：** 中長期のプロジェクトロードマップ、フェーズ別のアクション×レイヤー展開、Phase 1〜4のような段階的計画の全体像。「今どこにいて、次に何を、最終的にどうなるか」を1枚で示す場面に最適。

## Structure（構造）

コンテンツエリアを縦4段に分ける：フェーズヘッダー行、ゴール行、アクション行（レイヤー数だけ縦積み）、実現する価値行。横方向に4つのフェーズを等幅で並べる。フェーズヘッダーは矢印（chevron）形状。最終フェーズはアクセントカラーの塗りで強調する。左端に「フェーズ」「ゴール」「アクション」「実現する価値」等のラベル列を細く配置してもよい。

    structure:
      layout: phase-x-layer-roadmap
      rows:
        - name: "フェーズヘッダー（4つの右向き矢印）"
        - name: "ゴール行"
        - name: "アクション行（Layer A）"
        - name: "アクション行（Layer B）"
        - name: "実現する価値行"
      columns: 4
      final_phase:
        highlight: "アクセントカラー塗りつぶし"
      left_labels:
        width: 8%
        content: "行ラベル（フェーズ／ゴール／Layer A／Layer B／実現する価値）"

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| フェーズヘッダー | 各フェーズの上部 | 右向き矢印形状のバッジにフェーズ名＋期間を記載 |
| ゴール行 | フェーズヘッダーの直下 | 各フェーズで達成するゴールを2〜3行のテキストで記述 |
| アクション行 | ゴールの下（複数行） | レイヤー別（Layer A、Layer B等）のアクション項目を箇条書き |
| 実現する価値行 | 最下段 | 各フェーズ完了時に得られるビジネス価値を箇条書き |
| 行ラベル | 左端 | 各行が何を表すかのラベル（フェーズ／ゴール／アクション／価値） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-phase-roadmapのレイアウトで、Phase 1〜4の4フェーズを横並び矢印で表示し、各フェーズにゴール・Layer Aアクション・Layer Bアクション・実現する価値の4行を配置してください。最終フェーズはアクセントカラーで強調してください。デザインはスライドマスターに従ってください。」

**注意点：**
- フェーズ数は4が最適。3以下だと余白が空きすぎる、5以上は横幅が足りない
- 各セルのテキストは箇条書きで1〜3項目に抑える。詳細説明は別スライドに送る
- 最終フェーズを強調カラーで塗ることで「目指す姿」を視覚的に強調する
