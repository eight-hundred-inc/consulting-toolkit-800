# SLIDE-PATTERN: cover-gradient-text-bottom

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

---

## 概要

全面グラデーション背景スライド。タイトル・会社名・日付が左下エリアに縦積み、ロゴが右上に配置される表紙レイアウト。

---

## Structure

```
layout: full-bleed
zones:
  - role: background
    content: グラデーション背景（全面）
  - role: logo
    content: ロゴ（会社ロゴ・ブランドマーク）
  - role: text-block
    content: メインタイトル（大きく強調）＋会社名・証券コード＋日付（縦積み）
reading_order: 背景（全面）→ 右上（ロゴ）→ 左下（タイトル → 会社名・証券コード → 日付）
notes: タイトルは2〜3行まで対応（4行以上は文字サイズを下げる）
```

---

## Elements

| 要素 | 配置 | 役割 |
|---|---|---|
| グラデーション背景 | 全面 | ブランドカラーのグラデーション（装飾的背景） |
| ロゴ | 右上 | 会社ロゴ・ブランドマーク |
| メインタイトル | 左下エリア | 資料のタイトル（大きなフォント・太字） |
| 会社名・証券コード | タイトル直下 | 発行元情報 |
| 日付 | 最下 | 資料の作成・発表日 |

---

## Usage Guide

- グラデーション背景はスライドマスターのブランドカラーを使用する
- タイトルは2〜3行まで対応。4行以上は文字サイズを下げる
- ロゴはグラデーション背景とのコントラストを確保する
- テキストはグラデーション背景との視認性・コントラストを確保する
- 左下テキストブロックは左側の余白をスライド幅の5〜6%程度確保する
- 日付・会社名はタイトルより控えめなサイズで表示する
