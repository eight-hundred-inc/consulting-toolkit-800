このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

**パターン名：** three-col-large-icon-card
**概要：** 3列均等に並んだカードレイアウト。各カードは上部にアイコン＋見出しのヘッダーゾーン（アクセントカラー背景）と、下部に箇条書きリストのボディゾーンで構成される2段構造。
**適したシーン：** 3つの特徴・理由・強みを並列で比較・紹介するスライド。各項目に複数の根拠や詳細を箇条書きで添えたいとき。

## Structure（構造）

コンテンツエリアを3等分した列にカードを配置する。各カードは垂直方向に「ヘッダーゾーン（アイコン＋見出し）」と「ボディゾーン（箇条書きリスト）」の2つのセクションに分かれる。ヘッダーは塗り背景で強調し、アイコンはカード左上隅の折り返し装飾（ドッグイヤー）として表現することもある。

```yaml
layout: three-column-grid
zones:
  - role: cards
    content: カード ×3（均等配置、高さ揃え）
    elements:
      - role: header
        content: アイコン（またはドッグイヤー装飾）＋見出し（アクセントカラー背景）
      - role: body
        content: 箇条書きリスト 3〜4項目
reading_order: 左から右（カード）、各カードは上から下（ヘッダー→ボディ）
notes: 3列の各カードは高さを揃える。各カードの箇条書き項目数は同数に揃えるとバランスが良い。
```

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| カードヘッダー | 各カード上部 | アクセントカラー背景でカードのテーマを視覚的に区別 |
| アイコン | ヘッダー左（またはドッグイヤー） | カードテーマを象徴するアイコンまたは折り返し装飾 |
| 見出し | ヘッダー右（または中央） | カードのタイトル（2〜4語の短い見出し） |
| 箇条書きリスト | カードボディ | 各項目の詳細・根拠・特徴を3〜4つの箇条書きで記述 |
| カード背景 | カード全体 | 薄い塗り背景でカードの領域を明示（白または薄いアクセント色） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-three-col-large-icon-cardのレイアウトで、[テーマ]の3つの[特徴/理由/強み]をカード形式で並べてください。デザインはスライドマスターに従ってください。」

**注意点：**
- 3列の各カードの高さは揃えること（flex-stretchまたはmin-height指定）
- 各カードの箇条書き項目数は同じ数にすると整然としてバランスが良い
- ヘッダーの見出しは最大2行以内に収める
- アイコンの使用はデザインシステムの指定に従う（スライドマスターのアイコンセット参照）
- タイトルエリアとコンテンツエリアの間にサブタイトル行（1行のリード文）を入れることもある
