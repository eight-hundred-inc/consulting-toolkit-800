# SLIDE-PATTERN-section-divider

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスター（表紙・タイトル行・フッター・配色・フォント）はマスター側（branded-pptx / html-artifactテーマ等）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview
**パターン名：** section-divider
**概要：** セクション番号・章タイトル・リード文のみを大きな余白の中に配置する、章の切り替わりを示す中扉スライド。
**適したシーン：** プレゼンの各章の冒頭に挿入し、聴衆に話題の転換を明示的に伝えたいスライド。本文情報は持たず、次章の見出しと概要のみを伝える。
**対応する図解指名：** 中扉
**下流実装ヒント：** html-artifact: Section Divider ／ pptx: 全面レイアウトのセクション区切りスライド（通常のタイトル行・フッターは使わない専用マスター）

## Structure（構造）
このパターンは通常のタイトル行を持たない全面レイアウトの例外パターンである。スライド全体を1つの余白豊かな領域として扱い、中央よりやや上〜中央にセクション番号（アイキャッチ的な短いラベル）・章タイトル（大きな文字で1〜2行）・リード文（章の概要を1〜2行で説明する小さめの文章）を縦に積んで配置する。上下左右に大きな余白を確保し、情報量を最小限に絞ることで章の切り替わりを強く印象づける。

```yaml
layout: full-bleed-center-block
title_row: none（このパターンではタイトル行を使用しない）
zones:
  - role: content_block
    content: セクション番号（短い識別ラベル）→ 章タイトル（大きな文字、1〜2行）→ リード文（1〜2行、任意）を縦に積む
reading_order: セクション番号 → 章タイトル → リード文
notes: コンテンツブロックは垂直方向中央よりやや上、水平方向は左寄せまたは中央に配置する。上下左右に大きな余白を確保し、要素以外の装飾は最小限にする
```

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| セクション番号 | コンテンツブロック最上部 | この中扉がどの章に対応するかを示す短い識別ラベル |
| 章タイトル | セクション番号の直下 | これから始まる章の名称（このスライドの主役、最大フォントサイズ） |
| リード文 | 章タイトルの直下 | 章の概要・扱う論点を1〜2行で予告する補足テキスト |

## Usage Guide（AIへの使い方）

> 「SLIDE-PATTERN-section-dividerのレイアウトで、セクション番号［番号］、章タイトル［章タイトル］、リード文［章の概要1〜2行］を入れてください。デザインはスライドマスターに従ってください。」

**注意点：**
- 本文・図表・箇条書きなど詳細情報は一切載せない。中扉は「次に何を話すか」だけを伝えるためのスライドであり、情報を足すほど効果が薄れる。
- リード文は省略可能。章タイトルだけで十分に意図が伝わる場合は省いてよい。
- このパターンのみ通常のタイトル行（スライドマスターのTitle Area）を使用しない全面レイアウトである点に注意する。他のパターンと混在させる際はマスター切り替えが必要になる場合がある。
