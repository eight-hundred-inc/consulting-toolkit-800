# SLIDE-PATTERN-three-col-icon-card-body-text

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

**パターン名：** three-col-icon-card-body-text
**概要：** 3列のカードに大きめのイラスト/アイコン・見出し（2行）・CTAボタンを縦に積んだシンプルなカードレイアウト。カード全体が枠線で囲まれている。
**適したシーン：** 課題・問題点・ユーザーの悩みを3つ列挙するスライド。各カテゴリ・トピックをアイコンで視覚化してラベルボタンで分類したいとき。

## Structure（構造）

コンテンツエリアの上部にリード文を置き、その下に3つのカードを横並びで配置する。各カードは大きなイラスト/アイコンエリア・本文見出し（2行）・カテゴリラベルボタンで構成される。カード内に箇条書きや説明テキストはなく、視覚的なシンプルさを重視する。

```yaml
layout: three-column-grid
zones:
  - role: lead
    content: リード文（2行まで）
  - role: cards
    content: カード（枠線付き）×3、横並び。各カードはイラスト/アイコン（大）＋見出し（2行、中央揃え）＋カテゴリラベルボタン
reading_order: 上（リード文）→ 下（カード、左→右）
notes: カードは等幅・等間隔で高さを揃える。カテゴリラベルボタンはカード最下部に固定配置する。
```

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| リード文 | カードグループ上部 | スライド全体の補足説明・文脈を示すテキスト（2行まで） |
| イラスト/アイコン | カード上部中央、大きめ | 各カードのテーマを視覚的に示す大型イラスト（枠線なし） |
| 見出し（2行） | イラスト直下、中央揃え | カードの主題を2行で示すテキスト（改行して読みやすく） |
| カテゴリラベルボタン | カード最下部 | 各カードの分類・カテゴリ名をボタン形式で表示（角丸、ブランドカラー） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-three-col-icon-card-body-textのレイアウトで、営業における3つの課題（各課題のアイコン・課題説明・カテゴリラベル付き）を提示するスライドを作成してください。デザインはスライドマスターに従ってください。」

**注意点：**
- イラスト/アイコンはカードの上部2/3程度を占める大きなサイズにする
- 見出しは2行で改行し、中央揃えにする
- カテゴリラベルボタンは角丸を強めに（pill shape）してカード下部に固定配置する
- カードには細い枠線を適用する（ボーダー付きカード）
- リード文は2行以内に収め、スライドの状況説明に使う
- 3カードは等幅・等間隔で横並びにし、カード高さを揃える
