# SLIDE-PATTERN-chart-with-commentary

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスター（表紙・タイトル行・フッター・配色・フォント）はマスター側（branded-pptx / html-artifactテーマ等）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview
**パターン名：** chart-with-commentary
**概要：** 左3分の2にチャート（棒・ウォーターフォール等）、右3分の1に読み取りポイントの箇条書きを置く構成。
**適したシーン：** 1つの図表を主役に、そこから言えることを言葉で添えるとき。試算・PL推移・件数構成などの定量スライド。図と示唆を1枚で対応させたい場合。
**対応する図解指名：** チャート＋コメント
**下流実装ヒント：** html-artifact: 図版（棒/ウォーターフォール等）＋ Insight Callout ／ pptx: ネイティブチャート＋テキストボックス（右に番号付き or ●の解説）

## Structure（構造）
上部に全幅のリード文（任意）を置き、その下を左右に分割する。左3分の2をチャートエリア、右3分の1をポイント解説にあてる。解説は番号付きまたは●の箇条書きで3〜4件、各項目はチャート上の特徴（変化点・最大値・差分など）に対応させる。

```yaml
layout: two-third-chart-plus-notes
zones:
  - role: lead
    content: 全幅の要約文（1文・任意）
    optional: true
  - role: chart
    width: 66%
    content: チャートエリア（棒・ウォーターフォール・積み上げ等）
  - role: commentary
    width: 33%
    content: 読み取りポイントの箇条書き（番号または●）3〜4件
reading_order: 上（リード文）→ 左（チャート）→ 右（解説）
notes: 各解説項目はチャート上の特徴（変化点・最大値・差分など）に対応させる
```

## Elements（各要素の役割）
| 要素 | 配置 | 役割 |
|------|------|------|
| リード文 | 上部・全幅 | チャートから言える結論を1文で示す（任意） |
| チャートエリア | 左・約2/3 | 主役の図表（棒・ウォーターフォール・積み上げ等） |
| ポイント解説 | 右・約1/3 | 図表の読み取りどころを番号付き／●で3〜4件 |
| 解説マーカー | 各解説項目の先頭 | 番号または●で項目を区切る |

## Usage Guide（AIへの使い方）
プロンプト例：「左にウォーターフォールチャート、右に読み取りポイントを番号付きで3つ。チャートは○○の内訳を可視化し、右の各ポイントはチャート上の変化点に対応させて。」

注意点：
- チャートは1枚に1つ。複数の図を並べたい場合は別パターンにする。
- 右の解説はチャートの特徴に対応させ、図と無関係な一般論を書かない。3〜4件に絞る。
- チャート化するデータが未確定な段階では、チャートエリアをプレースホルダにして解説だけ先に埋めてよい。
