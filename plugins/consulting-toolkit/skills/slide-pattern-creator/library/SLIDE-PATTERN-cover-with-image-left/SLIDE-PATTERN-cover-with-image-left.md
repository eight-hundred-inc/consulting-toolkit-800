# SLIDE-PATTERN-cover-with-image-left

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

**パターン名：** cover-with-image-left
**概要：** 左半分（約45%）に画像、右半分（約55%）にカテゴリラベル・メインタイトル・説明文・日付などのメタ情報を縦に並べる表紙レイアウト。
**適したシーン：** 提案書・レポート・プレゼンの表紙、ビジュアルを前面に出したカバースライド

## Structure（構造）

スライド全体（タイトルエリアなし）を左右2分割で使うカバーレイアウト。左に画像、右にテキスト情報を縦に積み上げる。

```yaml
layout: full-slide-two-column (no title area)
zones:
  - role: image
    width: 45%
    content: 画像プレースホルダー（フル高さ）
  - role: text
    width: 55%
    content: カテゴリラベル（小・控えめ）＋メインタイトル（大・太字）＋説明文（1〜2行）＋区切り線＋メタ情報（日付・著者・会社名、控えめ）
reading_order: 左（画像）→ 右（カテゴリラベル → メインタイトル → 説明文 → 区切り線 → メタ情報）
notes: タイトルエリアを使わない全画面レイアウト。右カラムの各要素は縦に積み上げて配置する
```

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 画像プレースホルダー | 左カラム（高さ100%） | 表紙の視覚的インパクトを担う写真・イラスト領域 |
| カテゴリラベル | 右カラム上部 | 資料の種類・カテゴリを小さく表示（例：提案書・報告書） |
| メインタイトル | カテゴリラベルの下 | プレゼンのメインタイトルを大きく太く表示 |
| 説明文 | タイトルの下 | 1〜2行の概要説明 |
| 区切り線 | 説明文の下 | テキストブロックとメタ情報の視覚的分離 |
| メタ情報 | 区切り線の下 | 日付・会社名・担当者などを小さく表示 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-cover-with-image-leftのレイアウトで、タイトル「[タイトル]」、サブタイトル「[サブタイトル]」、日付「[日付]」の表紙スライドを作成してください。左側の画像は提供した画像を使用してください。デザインはスライドマスターに従ってください。」

**注意点：**
- このパターンはSlide Frameのタイトルエリアを使わない（スライド全体がコンテンツ）
- 左の画像はスライドの高さいっぱいに広がるようにすること
- 右のテキストは上揃えより縦中央揃えの方がバランスが取りやすい
- メタ情報（日付・会社名）は省略してもよい
