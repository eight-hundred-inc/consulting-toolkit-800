# SLIDE-PATTERN-three-stage-circle-flow
このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。タイトルエリアを除いたコンテンツ領域のみを対象とし、スライド全体のテンプレートではありません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

3つの大きな円形アイコン（ステージ名ラベル付き）を矢印で横に連結し、各円の下に見出し・サブタイトル・区切り線・箇条書きを配置するステージ型フローレイアウト。プロセスの3段階・3フェーズ・3ステップを順序立てて説明する際に適している。

- **適用場面:** プロセス解説、導入手順、ロードマップ概要
- **情報密度:** 中（段階ごとに3項目の詳細を内包）
- **方向性:** 左から右への時系列的な流れの可視化

## Structure

3つの大きな円形アイコンを矢印で横に連結し、各円の下に見出し・サブタイトル・区切り線・箇条書きを配置する。各ステージは均等幅で横並びになる。

```yaml
layout: three-stage-circle-flow
zones:
  - role: stage
    content: ステージ ×3（横並び、均等幅）
    elements:
      - role: circle-icon
        content: 円形コンテナ（枠面のある円）＋アイコン代替文字＋ステージ名ラベル
      - role: heading
        content: 見出し（中央揃え）
      - role: subtitle
        content: サブタイトル
      - role: divider
        content: 区切り線（短め、見出し・サブタイトルとリストの間）
      - role: bullet-list
        content: 箇条書き 3項目
  - role: arrow
    content: ステージ間の矢印 ×2（各ステージの間）
reading_order: 左から右（ステージ1 → 矢印 → ステージ2 → 矢印 → ステージ3）、各ステージは上から下（円形アイコン→見出し→サブタイトル→区切り線→箇条書き）
notes: 矢印は円の中央付近の高さに合わせて配置し、幅を保持して縮まないようにする。
```

## Elements

### ステージ円形アイコン
| 要素 | 説明 |
|------|------|
| 円形コンテナ | 円形の枠（アイコンを収める） |
| アイコン代替文字 | ステージのテーマを示すアイコン（実装時に画像やSVGアイコンに差し替え） |
| ステージ名 | 円の中または直下に表示するステージ名ラベル |

### テキストエリア（円の下）
| 要素 | 説明 |
|------|------|
| 見出し | 中央揃えでステージの主題を示す |
| サブタイトル | 見出しを補足する短いテキスト |
| 区切り線 | 見出し・サブタイトルとリストを分ける短い水平線 |
| 箇条書き | ステージの詳細を3項目で列挙 |

### 矢印
| 要素 | 説明 |
|------|------|
| 矢印文字 | 「→」 |
| 配置 | 円の中央付近の高さに合わせる |
| 挙動 | 幅を保持し縮まないよう固定する |

## Usage Guide

- 3ステージは「現状→移行→目標」「計画→実行→評価」のように対比構造を持たせると効果的
- 円形内のアイコン代替文字は実装時にアイコン画像やSVGアイコンに差し替える
- 各ステージの箇条書きは3項目で統一し、文字数も揃えるとバランスが良い
- 矢印は円の中央付近の高さに合わせて配置する
- ステージ名はステージ番号（STEP1など）とラベルを組み合わせると識別しやすい
