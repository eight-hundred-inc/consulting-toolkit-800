# SLIDE-PATTERN-two-col-icon-list-comparison

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

**パターン名：** two-col-icon-list-comparison
**概要：** コンテンツエリアを左右2カラムに分割し、各カラムに「カラーアイコン＋テキスト」のリストを縦に並べた比較レイアウト。コンテンツエリア上部にリード文を1〜2行置き、2カラムのそれぞれに見出しと3〜4項目のアイコン付きリストを配置する。
**適したシーン：** 原因と対策の比較、現状と施策の対応、AとBの観点比較など、2つの観点からそれぞれ複数の要素を並べて示したいスライド。

## Structure（構造）

コンテンツエリアの上部にリード文（中央揃えの1〜2行テキスト）を置き、その下を左右2等分のカラムに分割する。各カラムはコンテナで囲み、上部に見出し、下部にアイコン（正方形）＋テキストのリストアイテムを縦積みで並べる。左右カラムの間には矢印などの繋ぎ要素を置いてもよい。

```yaml
layout: two-column
zones:
  - role: lead
    content: リード文（中央揃え、1〜2行）
  - role: left-column
    width: 50%
    content: カラムコンテナ（見出し＋アイコン付きリスト 3〜4項目）
  - role: connector
    content: 中央コネクター（矢印等、任意）
  - role: right-column
    width: 50%
    content: カラムコンテナ（見出し＋アイコン付きリスト 3〜4項目）
reading_order: 上（リード文） → 左 → 右
notes: 左右カラムの項目数は揃える（3〜4項目が最適）。中央コネクターは省略可。
```

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| リード文 | コンテンツエリア最上部・中央揃え | スライドのテーマや2カラムの文脈を簡潔に説明する1〜2行の導入テキスト |
| カラムボックス | 左右の各カラム | 各グループをまとめるコンテナ（左右で見た目を変えると区別しやすい） |
| カラム見出し | 各カラムボックスの上部・中央揃え | そのカラムの内容を端的に示すラベル。強調して表示する |
| アイコン | 各リストアイテムの左端 | アイテムの内容を視覚的に補足する正方形のアイコン画像またはプレースホルダー |
| アイテムテキスト | アイコンの右横 | 各要素の説明テキスト。1〜2行で収める |
| リストアイテム区切り線 | アイテム間 | 各リストアイテムを細い横線で区切り可読性を高める |
| 中央コネクター | 左右カラムの中間 | 左から右への因果・流れを示す三角矢印や記号（省略可） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-two-col-icon-list-comparisonのレイアウトで、社会のデジタル化の流れ（左カラム）と企業が取り組むべきデジタル施策（右カラム）をアイコン付きリストで比較するスライドを作成してください。デザインはスライドマスターに従ってください。」

**注意点：**
- 左右カラムは同じ項目数で揃えると見栄えが良い（3〜4項目が最適）
- 左右カラムは見た目を変えると対比が明確になる
- アイコンは意味のある絵柄を選び、左右で見た目を変えるとカラムの違いが伝わりやすい
- リード文は2行以内に収め、カラムのスペースを圧迫しないようにする
