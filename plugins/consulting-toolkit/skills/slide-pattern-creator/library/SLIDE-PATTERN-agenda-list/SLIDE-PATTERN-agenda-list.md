# SLIDE-PATTERN-agenda-list

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

**パターン名：** agenda-list
**概要：** 章立ての目次リストをコンテンツエリア中央に配置し、上下を細い水平線で挟むシンプルな目次レイアウト。現在進行中のセクションは黒テキスト、それ以外はグレーアウトで表示することで「今どこか」が一目でわかる。
**適したシーン：** 冒頭のアジェンダ提示、章の切り替わりごとに繰り返し挿入するナビゲーションスライド、資料全体の構造を示したいプレゼンテーション。

## Structure（構造）

コンテンツエリアの上下に細い水平線を引き、その間に番号付きの章リストを縦に並べる。現在ページに該当する項目は通常表示、それ以外は抑えた表示にすることでカレント位置を明示する。

    layout: single-centered-list
    zones:
      - role: top-divider
        content: 上部の区切り線
      - role: list
        position: コンテンツエリア中央（ブロック自体は中央配置、テキストは左揃え）
        content: 番号付き章タイトルのリスト（形式: "{番号}. {章タイトル}"）5〜7項目
        emphasis: カレント章は通常表示、他の章は抑えた表示（グレーアウト）で統一する
      - role: bottom-divider
        content: 下部の区切り線
    reading_order: 上部区切り線 → 章リスト（上から下） → 下部区切り線
    notes: 章の数は5〜7項目が視覚的にちょうどよい。それ以上は two-column-toc の使用を検討する

## Elements（各要素の役割）

※ タイトル（H1）はスライドマスターの Slide Frame で定義されるため、この表には含めない。

| 要素 | 配置 | 役割 |
|---|---|---|
| 上部区切り線 | コンテンツエリア上端 | リストの開始位置を明示する薄いグレー水平線 |
| 章リスト | コンテンツエリア中央（縦方向） | 5〜7章程度の番号付き章タイトルを縦に列挙 |
| カレント章 | リスト内で現在ページに該当する項目 | Textカラーで通常表示し「今どこか」を示す |
| 非カレント章 | リスト内の他の項目 | Text Mutedカラーでグレーアウトし進捗感を表現 |
| 下部区切り線 | コンテンツエリア下端 | リストの終了位置を明示する薄いグレー水平線 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-agenda-listのレイアウトで、5〜7章の章タイトルを縦リストで表示し、現在のセクションを黒、それ以外をグレーで示してください。デザインはスライドマスターに従ってください。」

**注意点：**
- 章の数は5〜7項目が視覚的にちょうどいい。それ以上は two-column-toc の使用を検討する
- タイトルエリアには「アジェンダ」のみを置き、サブタイトルは基本的に不要
- 章切り替えごとに挿入する場合、カレント章の位置を毎回変えて使う
