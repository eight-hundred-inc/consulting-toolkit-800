## Overview

**パターン名：** diagram-with-callout
**概要：** コンテンツエリアを左右2カラムに分け、左に3ボックスの横フロー図＋補足箇条書き、右に「答え／結論」のハイライトテキスト＋番号付き例示リストを配置する、図解＋回答型レイアウト。
**適したシーン：** 「〜とは何か？」の問いに対して、左で構造・仕組みを図解し、右で具体的な答えと例示を並べるスライド。分析結果を提示する調査報告書の見開き構造にも使える。

## Structure（構造）

コンテンツエリアを縦の点線で左右約55%：45%に分割。左側は上段に3つの角丸ボックスを横並びにしたフロー図（矢印付き）、下段に3項目の解説箇条書き。右側は上部にアクセントカラーの結論文（2〜3行）、下部に番号付き円と例示テキストのリスト。

```yaml
layout: two-column-diagram-and-callout
zones:

    width: 55%
    content: 上段に3ボックス横フロー図（矢印つなぎ）、下段に3項目の解説箇条書き
  - role: right
    width: 45%

reading_order: 左（フロー図 → 解説箇条書き）→ 右（結論文 → 例示リスト）
notes: 左右エリアは区切り線（点線）で分割する
```

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-diagram-with-calloutのレイアウトで、左に3ボックスの横フロー図と箇条書き解説、右に結論文と番号付き例示3項目を配置してください。デザインはスライドマスターに従ってください。」

**注意点：**
- 左のフロー図は3ボックスが最適。4ボックス以上はfour-step-flowを使う
- 右の結論文は必ずアクセントカラー（グリーンまたはブルー）で強調する
- 右下の例示は「本プロジェクトを通じて具体化」のような小さな注記を添えると仮説の位置づけが伝わる
