# SLIDE-PATTERN-timeline-gantt

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。

## Overview

**パターン名：** timeline-gantt
**概要：** 横軸に週または月の時間軸を持ち、縦軸にタスク・ワークストリームを配置するガントチャート形式のスケジュール表。マイルストーンや集中討議などのイベントは縦の点線＋ラベルで強調し、上部にキックオフ・経営会議などの大きなマイルストーンを配置する。
**適したシーン：** プロジェクトのスケジュール提示、複数ワークストリームの並行進行の可視化、フェーズ移行のマイルストーン共有、実行計画の週次単位の落とし込み。

## Structure（構造）

コンテンツエリアの左に固定幅（20%程度）でタスク名を縦列、右に横軸のガントチャートエリア（80%）。横軸は等幅の週または月マス（5〜8列）で区切られる。各タスクは対応するマスに角丸バー（塗り分けOK）を描画。マイルストーンは縦のアクセントカラー線＋▼マーカー＋上部ラベルで表現。上部に大きなフェーズラベル（初期仮説構築・GD構築 等）のヘッダーバンドを配置することもある。

    structure:
      layout: task-x-timeline-gantt
      left_column:
        width: 20%
        content: "タスクID＋タスク名（縦列）"
      timeline_area:
        width: 80%
        header: "時間軸（週または月ラベル）"
        upper_band: "フェーズラベル（オプション）"
        rows: "各タスクのバー配置行"
      milestones:
        style: "縦のアクセントカラー線＋▼＋ラベル"
        placement: "重要な区切り日に配置"
      task_bar:
        style: "角丸矩形（グレー塗り、アクセントカラー塗り、点線輪郭など複数種類あり）"

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| ID列 | 左端 | タスク番号（A-1, B-1 等） |
| タスク名列 | ID列の右 | タスクの正式名称 |
| フェーズヘッダー | タイムライン上部の帯 | 大まかな期間の性格を示すラベル（初期仮説構築・構築 等） |
| 時間軸ラベル | タイムライン上部 | 5/7-8, 5/11w, 5/18w 等の週マーカー |
| マイルストーン | タイムライン中の縦線 | キックオフ・経営会議・集中討議など重要日をアクセントカラーで強調 |
| タスクバー | タイムライン中の横バー | タスクの開始〜終了を角丸矩形で表現。フェーズごとに塗り分け可 |
| バー内テキスト | タスクバー内 | 作業内容の一言（資料分析・仮説修正 等） |
| フェーズ矢印 | バー間 | バーからバーへの遷移を細い矢印で表現 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-timeline-ganttのレイアウトで、4つのタスクを縦に並べ、5/7〜6/19の7週間を横軸で表現するガントチャートを作ってください。キックオフ・集中討議・経営会議のマイルストーンを縦線で強調してください。デザインはスライドマスターに従ってください。」

**注意点：**
- タスク数は3〜6行が最適。7行以上は縦幅が窮屈になる
- マイルストーンは3個以内に絞り、それ以上のイベントは省略する
- バーの色分けはフェーズ（前半／後半、A/B）で意味を持たせる。単なる装飾には使わない
