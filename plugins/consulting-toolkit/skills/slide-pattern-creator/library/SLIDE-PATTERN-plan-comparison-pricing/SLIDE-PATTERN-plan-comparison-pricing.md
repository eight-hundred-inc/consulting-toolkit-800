# SLIDE-PATTERN-plan-comparison-pricing

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスター（表紙・タイトル行・フッター・配色・フォント）はマスター側（branded-pptx / html-artifactテーマ等）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview
**パターン名：** plan-comparison-pricing
**概要：** プラン名・タスク内訳・金額を横1行にまとめた行を複数プラン分、縦に並べる料金比較レイアウト。
**適したシーン：** 複数の提案プラン（松竹梅、A/B/C案等）を1枚で比較し、各プランの内容と金額を同時に見せたいスライド。
**対応する図解指名：** プラン比較＋金額
**下流実装ヒント：** html-artifact: Proposal Card＋金額列 ／ pptx: 3行×3列のテーブルまたはグリッド（左列プラン名カード・中央列タスク内訳・右列金額）

## Structure（構造）
コンテンツエリアはプラン数に応じた行（通常3行）で構成され、各行が1つのプランに対応する。各行は横方向に3つの領域に分かれる：左（プラン名カード。プラン名と短いキャッチコピー）、中央（タスク内訳。概要とタスクの範囲を箇条書きまたは短文で列挙）、右（金額。大きめのフォントで税込金額を表示）。行間には罫線を引いて各プランを区切る。

    layout: stacked-rows
    row_count: 3
    zones:
      - role: plan-name
        width: 20%
        content: プラン名カード（プラン名＋キャッチコピー）
      - role: task-breakdown
        width: 55%
        content: タスク概要＋タスク詳細（箇条書きまたは短文）
      - role: price
        width: 25%
        content: 金額（大きめ表示）＋金額注記
    reading_order: 上 → 下（行の並び）。各行内は左 → 中央 → 右
    notes: 行間に罫線を引いて各プラン行を区切る

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| プラン名カード | 各行の左端 | プランの名称と一言キャッチコピーを提示し、行の識別ラベルとなる |
| タスク概要 | 各行の中央上部 | そのプランに含まれる業務範囲の要約 |
| タスク詳細 | 各行の中央下部 | 業務範囲の内訳を箇条書き等で具体化する |
| 金額（大きめ数字） | 各行の右端 | プランの税込金額を最も目立つ大きさで表示し、比較の起点にする |
| 金額注記 | 金額の直下 | 「税込」「月額」等、金額の前提条件を補足する小さな注記 |

## Usage Guide（AIへの使い方）

> 「SLIDE-PATTERN-plan-comparison-pricingのレイアウトで、プラン［プランA/B/Cの名称・タグライン］、タスク内訳［各プランの業務範囲］、金額［各プランの税込金額］を入れてください。デザインはスライドマスターに従ってください。」

**注意点：**
- 行数（プラン数）は3〜4程度が上限。5行を超えると1行あたりの高さが不足し、タスク内訳が読みにくくなる。
- 金額は必ず他の要素より大きいフォントサイズにし、視線が最終的に金額へ着地するよう強弱をつける。
- おすすめプランがある場合は、そのプランのカード領域を強調色にする、または行全体を淡く塗るなどで差別化できる。
