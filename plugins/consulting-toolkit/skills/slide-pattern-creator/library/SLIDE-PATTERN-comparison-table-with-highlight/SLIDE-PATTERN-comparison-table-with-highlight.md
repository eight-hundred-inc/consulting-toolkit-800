# SLIDE-PATTERN-comparison-table-with-highlight

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。スライドマスターと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はスライドマスター（branded-pptx / html-artifactテーマ）で定義されるため、このファイルには含みません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

**パターン名：** comparison-table-with-highlight
**概要：** 複数のプラン・選択肢を列で比較する表形式レイアウト。ヘッダー行をアクセントカラーで塗りつぶし、特定セルの数値などをハイライトカラーで強調する。
**適したシーン：** 料金プラン比較、サービスプラン説明、仕様比較、機能の有無一覧など複数の選択肢を横断的に比較したいスライド。

## Structure（構造）

コンテンツエリアの中央に横幅いっぱいの比較テーブルを1つ配置する。テーブルは上部にヘッダー行、下部に各行を交互に区別したデータ行で構成される。各セルは縦方向中央揃えで、強調したいデータ（価格など）は強調表示する。

    layout: single
    zones:
      - role: header-row
        content: 列ヘッダー ×5（各列の項目名）。強調表示
      - role: data-rows
        content: データ行 ×3（各行：行ラベルセル＋データセル）。行を交互に区別
      - role: highlight-cells
        content: 強調セル（価格・数値など重要な値）
    reading_order: 上（ヘッダー行）→ 下（各データ行）、左（行ラベル）→ 右（データ）
    notes: 比較軸は4〜6列程度、行は3〜5行程度が目安。ハイライトする値は1〜2列に絞る

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| テーブルヘッダー行 | テーブル最上部 | 各列の項目名を表示。他の行と区別して強調する |
| 行ラベルセル（第1列） | 各データ行の左端 | 比較対象の名称（プラン名など）を表示。他の列と区別して表示する |
| 通常データセル | ヘッダー以外の各セル | テキストや数値などのデータを表示。行を交互に区別する |
| ハイライトセル | 強調したいデータセル | 価格・数値など重要な値を強調表示する |
| テーブル外枠・罫線 | セル境界 | セルを区切り、可読性を高める |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-comparison-table-with-highlightのレイアウトで、3つのサービスプランを料金・対応範囲・納期・おすすめ対象の4軸で比較するスライドを作成してください。デザインはスライドマスターに従ってください。」

**注意点：**
- 列数が増えすぎると各セルが狭くなるため、比較軸は4〜6列程度に収める
- ハイライトする値（価格・数値）は1〜2列に絞り、強調が分散しないようにする
- テーブルの縦幅が大きくなる場合は行の高さを均等に調整してコンテンツエリアに収める
- ヘッダー背景色はスライドマスターのアクセントカラーに合わせる
