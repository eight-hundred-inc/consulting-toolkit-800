# SLIDE-PATTERN: circle-node-step-flow

## Overview

円形ノードを横一列に並べ、細い線で繋いだ水平フロー図。最後のノードを塗りつぶし（ゴール・完了）で強調するステッププロセス表現。

## 適したシーン

採用フロー・申請プロセス・審査ステップなど、5〜6段階の工程を丸みのある図解で示したいとき。

## Structure

```yaml
layout: horizontal-flow
zones:
  - role: lead
    content: リード文（フロー全体の説明、1〜2行、任意）
  - role: step-nodes
    content: 円形ノード ×5（横一列・等間隔、細い線で接続）。各ノード内にステップ名（1〜3語）＋補足テキスト（任意）
  - role: goal-node
    content: 最終ノード（最右端、塗りつぶしで強調）
  - role: footnote
    content: 注記（補足・注意事項、任意）
reading_order: 上（リード文）→ 左から右（ステップ順）→ 下（注記）
notes: 円は5〜7個が適切（8個以上は円が小さくなり文字が読みにくい）。最終ノードの塗りつぶしは「ゴール」「合格」を暗示するため、ネガティブな最終ステップには不向き
```

## Elements

| 要素 | 配置 | 役割 |
|---|---|---|
| ステップ円 | 横一列・等間隔 | 各ステップを表す円形ノード（ボーダーのみ） |
| ゴール円 | 最右端 | 最終ステップ（塗りつぶしで強調） |
| ステップ名 | 各円の内部 | ステップの名称（1〜3語） |
| 補足テキスト | 各円の内部・名称下 | 注記（所要期間・補足等、省略可） |
| 繋ぎ線 | 各円の間 | フローの方向を示す細い水平線 |
| リード文 | コンテンツ上部 | フロー全体の説明文（1〜2行） |
| 注記 | コンテンツ下部 | 補足・注意事項 |

## Usage Guide

- 円は5〜7個が適切。8個以上は円が小さくなり文字が読みにくい
