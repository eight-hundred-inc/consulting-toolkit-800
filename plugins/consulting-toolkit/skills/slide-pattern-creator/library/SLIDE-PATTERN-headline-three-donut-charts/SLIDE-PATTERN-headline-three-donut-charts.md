# SLIDE-PATTERN-headline-three-donut-charts
このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。タイトルエリアを除いたコンテンツ領域のみを対象とし、スライド全体のテンプレートではありません。プレビュー .html は生成当時の見本であり非規範。見た目は選択マスターが決めます。

## Overview

コンテンツエリア上部に大きな見出し2行（キーメッセージ）を配置し、下部に3つのドーナツグラフを横並びで表示する数値インパクトレイアウト。重要なメッセージを文字で強調しながら、その根拠となる3つの数値指標をグラフで視覚化したい場合に適している。

- **適用場面:** エグゼクティブサマリー、成果発表、数値インパクト訴求
- **情報密度:** 中（メッセージ＋グラフ3点の明快な構成）
- **方向性:** 「言葉のインパクト＋数値の裏付け」の組み合わせ

## Structure

コンテンツエリアを上下に分割する。上部に見出しエリア、下部に3つのドーナツチャートを横並びで配置する。

```yaml
layout: headline-three-donut-charts
zones:
  - role: headline
    content: エリアラベル＋大見出し2行（キーメッセージ＋補足メッセージ）
  - role: divider
    content: 区切り線
  - role: charts
    content: ドーナツチャートカード ×3（横並び。各カード＝ドーナツ図＋中央数値＋中央ラベル＋見出し＋説明テキスト＋凡例）
reading_order: 上（見出し）→ 下（グラフ3点、左→右）
notes: 3つのドーナツは同じ指標種別（達成率・割合・スコア等）で統一する
```

## Elements

### 上部見出しエリア
| 要素 | 説明 |
|------|------|
| エリアラベル | 「KEY MESSAGE」等の小文字ラベル |
| 大見出し（1行目） | 太字の大きな見出しテキスト |
| 大見出し（2行目） | 太字の大きな見出しテキスト |
| 区切り線 | 見出しとグラフエリアを分ける細い横線 |

### 下部グラフエリア（3セル）
| 要素 | 説明 |
|------|------|
| ドーナツSVG | 2分割配色の円グラフ |
| 中央数値 | 太字の強調数値 |
| 中央ラベル | 小さめの補足ラベル |
| 見出し | 太字の見出し |
| 説明テキスト | 中央寄せの説明テキスト |
| 凡例 | データ名と残りを示す横並びの凡例 |

## Usage Guide

- 見出しは「〇〇が△△%改善」「□□において◇◇を達成」のように、具体的な成果を短く記述する
- 3つのドーナツは同じ指標種別（達成率、割合、スコアなど）で統一するとわかりやすい
- 見出しは2行に収まる文字量を基準とし、文字数が多い場合はやや小さくして調整する
- 説明テキストは1行に収める。2行になる場合はより小さいサイズに調整する
- 凡例はデータ名と残り（「その他」等）の2項目で構成する
